# Sales Report Dashboard - Power BI
 
A Power BI dashboard analyzing sales data across multiple products, categories, and countries.
 
## About
 
This is my first Power BI project. I did a crash course from YouTube to get hands-on experience with Power BI.
 
## Overview
 
This interactive dashboard provides insights into sales performance, built as part of a Power BI crash course project.
 
## Key Metrics
 
- **12** Customers
- **100** Orders
- **102.5K** Total Sales
 
## Dashboard Features
 
- **Sales by Year & Quarter** — Line chart tracking sales trends from 2024 to 2026
- **Total Sales by Product** — Bar chart comparing sales across 10 products (Dell XPS 15, ThinkPad X1, Galaxy S24, etc.)
- **Total Sales by Categories** — Donut chart showing distribution across Laptop, Smartphone, Tablet, Smart Home, and Accessory categories
- **Interactive Filters** — Filter by Year, Country (China, Germany, India, United States), and Product Category
 
## Data Sources
 
- `customers.csv` — Customer information
- `orders.csv` — Order details
- `Practice.csv` — Practice dataset
 
## Tools Used
 
- Microsoft Power BI Desktop
